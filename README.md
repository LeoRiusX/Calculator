# Calculator in Python (Tkinter)

A simple calculator, using the tkinter.

## Features :
-> Support for basic arithmetic operations: +, -, *, /\
-> Ability to enter decimal numbers (.)\
-> C button: short press deletes the last character, long press clears the entire input\
-> Error checking: division by zero, incorrect operator sequences (++, **, etc.), empty input

## Сode information:

Numbers and operators are entered using the buttons\

Long press (>0.5 seconds) – clear all input
